# 🔍 Algorithm Visualizer

An interactive web application that brings algorithms to life through step-by-step visual animations. Built with a React frontend and Node.js backend, it helps students and developers build intuition for how algorithms work under the hood.

---

## ✨ Features

- **Real-time Visualizations** — Watch algorithms execute step by step with animated transitions
- **Multiple Algorithm Categories** — Sorting, pathfinding, searching, and more
- **Adjustable Speed Controls** — Step through slowly to understand, or speed up to compare performance
- **Clean, Responsive UI** — Works across desktop and mobile browsers
- **Frontend + Backend Architecture** — React-powered UI with a Node.js API layer

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v16 or higher
- npm v8 or higher

### 1. Clone the Repository

```bash
git clone https://github.com/pavankalyan7127/Algorithm-Visualizer.git
cd Algorithm-Visualizer
```

### 2. Start the Backend

```bash
cd Backend
npm install
npm start
```

The backend server will start at `http://localhost:5000` (or as configured).

### 3. Start the Frontend

Open a new terminal:

```bash
cd Frontend
npm install
npm start
```

The React app will open at `http://localhost:3000`.
